# Getting Started [Get started in just a few lines of code.]

## Overview

`@bgd-labs/toolbox` is a utility package that includes a wide range of features useful for everyday work in web3.

`@bgd-labs/cli` as the name suggests is a cli package that can be executed as standalone via `npx @bgd-labs/cli`

## Installation

:::code-group

```bash [npm]
npm i @bgd-labs/toolbox
```

```bash [pnpm]
pnpm i @bgd-labs/toolbox
```

```bash [bun]
bun i @bgd-labs/toolbox
```

:::
